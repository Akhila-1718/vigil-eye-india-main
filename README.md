# vigil-eye-india-main
The main aim of this project is to use Artificial Intelligence and Deep Learning with CCTV cameras to automatically detect suspicious activities and fear expressions in crowded public places to prevent women and child kidnapping incidents.
This system continuously monitors CCTV footage, identifies humans, analyzes emotions such as fear or distress, detects abnormal or forceful actions like dragging or pulling, and immediately sends an alert to authorities to take quick action.
